# 🚀 빠른 시작 가이드

## 1단계: 설치

### 방법 A: 자동 설치 (권장)
```powershell
# setup.bat 파일을 더블클릭하거나 명령 프롬프트에서 실행
setup.bat
```

### 방법 B: 수동 설치
```powershell
# 필요한 라이브러리 설치
pip install -r requirements.txt
```

## 2단계: 실행

### 방법 A: 배치 파일 사용
```powershell
# run.bat 파일을 더블클릭
run.bat
```

### 방법 B: 직접 실행
```powershell
python main.py
```

## 3단계: 사용하기

### 시스템 트레이 확인
- 프로그램이 실행되면 시스템 트레이(작업 표시줄 오른쪽 하단)에 파란색 아이콘이 나타납니다
- 아이콘이 보이지 않으면 화살표(^)를 클릭하여 숨겨진 아이콘을 확인하세요

### 테스트하기
1. 아무 텍스트를 복사합니다 (Ctrl+C)
2. 메모장이나 다른 프로그램을 엽니다
3. Ctrl+V를 눌러 붙여넣기를 시도합니다
4. 확인 팝업이 나타나면 성공! ✅

### 설정 열기
- 트레이 아이콘을 우클릭 → "Settings" 클릭

## 주요 기능 사용법

### 📝 화이트리스트 추가
특정 프로그램에서는 확인 없이 바로 붙여넣기하려면:

1. 설정 창 열기
2. "✓ Whitelist" 탭 선택
3. 프로세스 이름 입력 (예: notepad.exe, code.exe)
4. "+ Add" 버튼 클릭

**팁**: 프로세스 이름을 모르는 경우
- 작업 관리자 (Ctrl+Shift+Esc) 열기
- "세부 정보" 탭에서 프로그램 이름 확인

### 🎛️ 모니터링 설정
텍스트 또는 이미지만 선택적으로 모니터링:

1. "📋 Monitoring" 탭 선택
2. "Text Content" 또는 "Image Content" 토글 스위치 조절

### 🎨 투명도 조절
팝업 창의 투명도를 조절하려면:

1. "🎨 Appearance" 탭 선택
2. 슬라이더를 좌우로 움직여 투명도 조절
3. 변경 사항은 즉시 적용됩니다

## 종료 방법

1. 시스템 트레이 아이콘 우클릭
2. "Exit" 선택

**또는** Ctrl+C를 눌러 콘솔 창에서 종료

## 문제 해결

### ❌ "Python is not installed" 오류
- Python 3.8 이상을 설치하세요: https://python.org
- 설치 시 "Add Python to PATH" 옵션을 체크하세요

### ❌ 팝업이 나타나지 않음
1. 프로그램이 실행 중인지 확인 (트레이 아이콘 확인)
2. 설정에서 모니터링이 활성화되어 있는지 확인
3. 해당 프로그램이 화이트리스트에 없는지 확인

### ❌ 특정 프로그램에서 작동하지 않음
- 일부 시스템 프로그램은 관리자 권한이 필요합니다
- 명령 프롬프트를 관리자 권한으로 실행한 후 프로그램 실행:
  ```powershell
  # 관리자 권한 명령 프롬프트에서
  python main.py
  ```

### ❌ 키보드 후킹이 차단됨
- 백신 프로그램이나 보안 소프트웨어가 차단할 수 있습니다
- 예외 목록에 Python 또는 프로그램 폴더를 추가하세요

## 고급 사용법

### 자동 시작 설정
Windows 시작 시 자동으로 실행하려면:

1. `Win + R` 키를 눌러 "실행" 창 열기
2. `shell:startup` 입력 후 엔터
3. 시작 프로그램 폴더가 열림
4. `run.bat` 파일의 바로가기를 이 폴더에 복사

### 설정 파일 직접 편집
고급 사용자는 `config.json` 파일을 직접 편집할 수 있습니다:

```json
{
    "monitor_text": true,        // 텍스트 모니터링
    "monitor_image": true,       // 이미지 모니터링
    "whitelist": [               // 화이트리스트
        "notepad.exe",
        "code.exe"
    ],
    "popup_opacity": 0.95,       // 투명도 (0.5 ~ 1.0)
    "theme": "dark",             // 테마
    "accent_color": "#3B82F6"    // 포인트 컬러
}
```
